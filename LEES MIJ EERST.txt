COMMAND PLATFORM STARTEN OP WINDOWS

1. Pak het ZIP-bestand uit.
2. Open de map command-platform-v24.
3. Dubbelklik op: Start Command Platform.bat
4. Er opent een zwart venster.
5. Daarna opent de app automatisch in je browser.
6. Laat het zwarte venster open tijdens gebruik.
7. Sluit het zwarte venster om de app te stoppen.

Werkt het niet?
Dan ontbreekt waarschijnlijk Python op de computer.
